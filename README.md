# Sales Analytics Dashboard (Power BI)

## 🔍 О проекте
BI-дашборд для анализа продаж, прибыли и рентабельности. Выявлены точки роста и убыточные зоны бизнеса.

## 🎯 Цель
- Анализ динамики продаж
- Выявление прибыльных сегментов
- Поиск убыточных заказов и зон риска

## 📊 Основные метрики
- **Выручка:** 1.1 млн
- **Прибыль:** 253.8 тыс
- **Рентабельность:** 23%
- **Средний чек:** 1.93 тыс

## 📈 Структура дашборда
1. **Overview** — KPI, динамика продаж
2. **Анализ** — сегменты, регионы, категории
3. **Проблемные зоны** — убыточные заказы, менеджеры

## 🔍 Ключевые инсайты
- 70% выручки даёт сегмент Consumer
- Пик продаж — март
- Есть убыточные заказы и менеджеры
  
## 💡 Рекомендации
- Оптимизировать логистику
- Ограничить скидки на убыточные категории
- Сфокусироваться на прибыльных сегментах

## 📥 Как скачать и открыть дашборд
1. В корне репозитория найдите файл `дашборд задание Арефьева.pbix`
2. Нажмите на него → откроется страница файла
3. Нажмите кнопку **View raw** — файл скачается
4. Откройте скачанный файл в **Power BI Desktop**
